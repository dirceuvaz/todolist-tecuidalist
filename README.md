# Te Cuida List

Gerenciador de tarefas Kanban offline para Windows.

100% local, multi-usuário, sem necessidade de internet ou contas online.

## Funcionalidades

- Quadro Kanban com 3 colunas (A Fazer, Em Andamento, Concluído)
- Múltiplos boards
- Múltiplos usuários no mesmo computador
- Dashboard com métricas e progresso
- Drag-and-drop por handle
- Temas escuro e claro
- Export e Import CSV
- Admin embutido para gerenciar usuários
- Instalador Windows NSIS com ícone personalizado

## Como instalar

Baixe o instalador na [página de releases](https://github.com/dirceuvaz/todolist-tecuidalist/releases) e execute.

Na primeira execução, o app guiará você pelo wizard de setup inicial.

## Como desenvolver

```bash
npm install
npm run electron    # modo desenvolvimento
npm run build       # gerar instalador
```
